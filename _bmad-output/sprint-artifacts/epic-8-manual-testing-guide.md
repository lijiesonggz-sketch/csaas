## Story 8.1: 同业爬虫源管理

### 验证步骤

#### 1. 访问管理后台
**步骤:**
1. 登录系统
2. 导航至: 管理后台 → 同业爬虫管理

**期望结果:**
- [ ] 页面正常加载，显示"同业爬虫源管理"标题
- [ ] 显示爬虫源列表（如有数据）
- [ ] 显示"添加源"和"测试爬取"按钮

#### 2. 添加爬虫源
**步骤:**
1. 点击"添加源"按钮
2. 填写表单:
   - 源名称: `测试源-拉勾网`
   - 源类型: `招聘网站`
   - URL: `https://www.lagou.com/gongsi/j1234.html`
   - 爬取频率: `60` 分钟
   - 选择器: `.job-list .item`
3. 点击"保存"

**期望结果:**
- [ ] 表单验证正常工作（必填项校验）
- [ ] 保存成功后返回列表页
- [ ] 新添加的源显示在列表中
- [ ] 状态显示为"正常"

#### 3. 测试爬取功能
**步骤:**
1. 在列表中选择一个源
2. 点击"测试爬取"按钮
3. 等待爬取完成

**期望结果:**
- [ ] 显示测试爬取弹窗
- [ ] 显示爬取进度
- [ ] 爬取完成后显示结果预览
- [ ] 显示成功/失败状态

#### 4. 编辑和删除源
**步骤:**
1. 点击源的"编辑"按钮
2. 修改爬取频率为 30 分钟
3. 保存
4. 点击"删除"按钮
5. 确认删除

**期望结果:**
- [ ] 编辑表单预填充现有数据
- [ ] 保存后数据更新
- [ ] 删除前显示确认对话框
- [ ] 删除后从列表中移除

**期望结果:**

#### 2. 验证并发控制

1. 创建多个测试源（5个以上）
2. 同时触发手动爬取
3. 观察任务执行顺序

**期望结果:**
- [ ] 任务按优先级排队执行
- [ ] 不会同时执行过多任务
- [ ] 任务日志显示正确的执行顺序

#### 3. 验证动态频率调整
**步骤:**
1. 编辑一个源，将频率从 60 分钟改为 15 分钟
2. 保存后观察任务调度

**期望结果:**
- [ ] 新频率立即生效
- [ ] 下次执行时间按新频率计算
