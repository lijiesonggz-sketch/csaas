```
Check if you specified a correct entity object and if it's connected in the connection options.
```

**已验证**:
- ✅ Tenant实体已在typeorm.config.ts中注册
- ✅ Organization和Tenant的关系定义正确

2. 尝试使用`autoLoadEntities: true`
3. 显式指定实体路径
