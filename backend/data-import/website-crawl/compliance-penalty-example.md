# 某城市商业银行数据安全管理违规处罚通报

## 被处罚机构
某城市商业银行股份有限公司

## 处罚原因
经查，该行在数据安全管理方面存在以下违规行为：

1. **客户敏感信息保护不到位**
   - 未建立完善的数据分类分级制度
   - 敏感客户信息访问控制机制缺失
   - 数据脱敏处理不充分

2. **数据治理体系不健全**
   - 未设立专门的数据治理委员会
   - 数据质量管理制度缺失
   - 数据安全责任制未落实

3. **违反监管规定**
   - 违反《银行业金融机构数据治理指引》第五条规定
   - 未按要求报送数据治理报告
   - 数据安全管理存在重大隐患

## 处罚决定
根据《中华人民共和国银行业监督管理法》第四十六条规定，对该行处以**50万元罚款**，并责令限期整改。

## 整改要求
1. 建立完善的数据分类分级制度（30日内完成）
2. 加强数据访问控制和权限管理（30日内完成）
3. 设立数据治理委员会，明确数据安全责任（15日内完成）
4. 制定数据安全管理制度和操作规程（30日内完成）
5. 定期开展数据安全审计和培训（持续执行）

## 政策依据
- 《银行业金融机构数据治理指引》
- 《中华人民共和国银行业监督管理法》
- 《商业银行信息科技风险管理指引》
